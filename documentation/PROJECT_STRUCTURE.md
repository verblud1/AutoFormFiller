# Структура проекта AutoFormFiller

## Обзор

Проект AutoFormFiller состоит из двух основных компонентов:
1. **Создатель JSON файлов** (`family_creator`) - для формирования и редактирования JSON файлов с данными семей
2. **Массовый обработчик** (`mass_processor`) - для автоматического заполнения форм в базе данных

## Структура каталогов

```
AutoFormFiller/
├── main.py                     # Главный исполняемый файл с лаунчером
├── PROJECT_STRUCTURE.md        # Этот файл документации
├── family_creator/             # Компонент создания JSON файлов
│   ├── __init__.py
│   ├── main.py                 # Основной запуск создателя JSON
│   ├── json_generator.py       # Генератор JSON файлов
│   └── gui.py                  # GUI компоненты создателя
├── mass_processor/             # Компонент массовой обработки
│   ├── __init__.py
│   ├── main.py                 # Основной запуск обработчика
│   ├── core.py                 # Ядро массового обработчика
│   └── processor.py            # Обработчик данных
├── utils/                      # Утилитарные функции
│   ├── __init__.py
│   ├── data_processing.py      # Обработка данных
│   ├── file_utils.py           # Работа с файлами
│   ├── validation.py           # Валидация данных
│   ├── excel_utils.py          # Работа с Excel файлами
│   └── family_processor.py     # Обработка данных семей
├── common/                     # Общие компоненты
│   └── gui_components.py       # Общие GUI компоненты
└── Installer/                  # Файлы установки (оригинальные)
    └── ...
```

## Описание компонентов

### family_creator/
Компонент для создания и редактирования JSON файлов с данными семей.

- `main.py`: Точка входа в приложение создателя JSON
- `json_generator.py`: Класс для генерации и управления JSON файлами
- `gui.py`: Графический интерфейс создателя JSON

### mass_processor/
Компонент для автоматического заполнения форм в базе данных на основе JSON файлов.

- `main.py`: Точка входа в приложение массового обработчика
- `core.py`: Основная логика массового обработчика
- `processor.py`: Обработчик данных для автоматизации

### utils/
Общие утилиты, используемые обоими компонентами:

- `data_processing.py`: Функции для обработки и очистки данных
- `file_utils.py`: Функции для работы с файлами и конфигурацией
- `validation.py`: Функции валидации данных
- `excel_utils.py`: Функции для работы с Excel файлами
- `family_processor.py`: Общие функции обработки данных семей

### common/
Общие компоненты интерфейса:

- `gui_components.py`: Базовые GUI компоненты для обоих приложений

## Зависимости

Проект использует следующие основные библиотеки:
- `customtkinter` - для графического интерфейса
- `selenium` - для автоматизации браузера
- `pandas` - для работы с Excel файлами
- `openpyxl` - для чтения Excel файлов
- `webdriver-manager` - для управления драйверами браузера

## Запуск

Для запуска приложения используйте команду:
```bash
python main.py
```

Это откроет лаунчер с выбором режима работы.

Для запуска конкретного компонента:
```bash
# Создатель JSON файлов
python main.py --mode creator

# Массовый обработчик
python main.py --mode processor
