# AutoFormFiller - Система автоматического заполнения форм

## Описание

AutoFormFiller - это комплексное решение для автоматического заполнения форм в базе данных на основе JSON файлов с информацией о семьях. Система состоит из двух основных компонентов:

1. **Создатель JSON файлов** - позволяет формировать и редактировать JSON файлы с данными семей
2. **Массовый обработчик** - автоматически заполняет формы в базе данных на основе JSON файлов

## Особенности

- Графический интерфейс на основе CustomTkinter
- Поддержка загрузки данных из Excel файлов (реестры, АДПИ)
- Автоматическое определение семей по ФИО
- Валидация данных перед сохранением
- Возможность ручного вмешательства при необходимости
- Поддержка автосохранения данных
- Поддержка скриншотов заполненных форм

## Требования

- Python 3.7+
- Chrome, Yandex или Chromium браузер
- База данных (доступ к localhost:8080)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd AutoFormFiller
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

Если файл requirements.txt отсутствует, установите следующие пакеты:
```bash
pip install customtkinter selenium pandas openpyxl python-dateutil webdriver-manager
```

## Запуск

### Лаунчер (рекомендуется)
```bash
python main.py
```

### Прямой запуск компонентов
```bash
# Запуск создателя JSON файлов
python main.py --mode creator

# Запуск массового обработчика
python main.py --mode processor
```

## Использование

### Создатель JSON файлов

1. Запустите приложение в режиме создателя
2. Загрузите JSON файл с существующими семьями или создайте новые
3. Заполните данные о семье (родители, дети, доходы, жилье, АДПИ)
4. Используйте вкладку "Автоопределение" для загрузки данных из реестров
5. Сохраните данные в JSON файл

### Массовый обработчик

1. Запустите приложение в режиме обработчика
2. Загрузите JSON файл с семьями
3. Настройте параметры автоматизации
4. Начните процесс обработки
5. При необходимости используйте ручное вмешательство

## Файлы конфигурации

- `family_creator_config.json` - конфигурация создателя JSON файлов
- `mass_processor_config.json` - конфигурация массового обработчика
- `processing_statistics.json` - статистика обработки

Файлы конфигурации хранятся в папке `config/`.

## Папки

- `autosave/` - автосохраненные данные
- `screenshots/` - скриншоты заполненных форм
- `logs/` - логи работы приложения
