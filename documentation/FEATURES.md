# Возможности системы автоматического заполнения форм

## Основные возможности

### 1. Работа с Google Sheets
- Закрашивание выполненных семей в Google Sheets зеленым цветом
- Проверка наличия зеленого или желтого цвета в ячейках перед закрашиванием
- Автоматическое сохранение и подстановка ID таблицы
- Автоматическое сохранение и подстановка названия листа
- Подтверждение данных пользователем перед использованием

### 2. Управление статусом окрашивания
- Проверка ключа `isColored` в JSON файлах
- Обновление статуса окрашивания после успешной операции
- Пропуск уже окрашенных семей при повторном запуске

### 3. Интерактивный выбор действий
- Проверка семей с уже установленными цветами
- Возможность выбора: "да", "нет", "выбрать конкретные семьи"
- Подтверждение данных пользователем с возможностью корректировки

### 4. Конфигурационное управление
- Сохранение ID таблицы в конфигурационный файл
- Сохранение названия листа в конфигурационный файл
- Автоматическая подстановка сохраненных значений
- Возможность переопределения значений пользователем

### 5. Обработка семей
- Поиск семей по ФИО матери или отца
- Проверка наличия цвета в ячейках
- Интерактивная обработка семей с уже установленным цветом
- Обновление JSON файлов с информацией о статусе окрашивания

## Примеры использования

### Сохранение ID таблицы
```bash
python examples/save_spreadsheet_id.py
```

### Сохранение названия листа
```bash
python examples/save_sheet_name_example.py
```

### Интерактивная обработка семей
```python
from utils.google_sheets_handler import interactive_check_existing_colors_and_highlight_with_auto_config

success = interactive_check_existing_colors_and_highlight_with_auto_config(
    credentials_file="your_credentials.json",
    json_file_path="path/to/families.json",
    sheet_name="АСП_Многодетные"
)
```

После первого запуска, ID таблицы и название листа будут сохранены в конфигурацию и использоваться автоматически в последующих запусках.
