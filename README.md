# AutoFormFiller

Система автоматического заполнения форм и управления семьями с интеграцией Google Sheets

## Основные возможности

- Автоматическое заполнение форм для семей
- Интеграция с Google Sheets для закрашивания выполненных семей
- Проверка существующих цветов в таблице перед закрашиванием
- Автоматическое сохранение и подстановка ID таблицы и названия листа
- Интерактивный выбор действий для семей с уже установленным цветом
- Обновление статуса окрашивания в JSON файлах

## Новая функциональность: автоматическое сохранение и подстановка ID таблицы

Система теперь автоматически сохраняет ID Google-таблицы и названия листов в конфигурационный файл `config/app_config.json`. При последующих запусках ID и название подставляются автоматически с возможностью проверки и корректировки пользователем.

Пример использования автоматического сохранения:

```python
from utils.config_manager import get_default_config_manager
from utils.google_sheets_handler import get_both_ids_with_confirmation

# Получение ID таблицы и названия листа с автоматической подстановкой
config_manager = get_default_config_manager()
spreadsheet_id, sheet_name = get_both_ids_with_confirmation("АСП_Многодетные", config_manager)
```

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Положите файл учетных данных Google API в корень проекта

## Использование

Запустите основной лаунчер:
```bash
python main.py
```

## Структура проекта

- `utils/` - служебные модули, включая `google_sheets_handler.py` и `config_manager.py`
- `launcher/` - модули для запуска системы
- `examples/` - примеры использования новой функциональности
- `config/` - конфигурационные файлы (создаются автоматически)

## Основные функции

1. **Автоматическая проверка цветов**: система проверяет, есть ли уже установленный цвет в ячейках перед закрашиванием
2. **Интерактивный выбор**: при обнаружении окрашенных ячеек пользователю предлагается выбрать: "да", "нет" или "выбрать конкретные семьи"
3. **Автоматическое обновление статуса**: после закрашивания в JSON файлах обновляется статус `isColored` для соответствующих семей
