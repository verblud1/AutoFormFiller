# AutoFormFiller

Приложение для автоматического заполнения форм в веб-системе.

## Установка

1. Установите Python 3.8 или выше с официального сайта: https://python.org

2. Скачайте файлы приложения в отдельную папку

3. Установите зависимости:
```cmd
pip install -r requirements.txt
