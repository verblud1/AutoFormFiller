# Инструкция по установке для сотрудников

## Поддерживаемые системы:
- ✅ Windows 7/8/10/11
- ✅ RED OS 7/8

## Установка:

1. **Скачайте архив** с программой
2. **Распакуйте** в любую папку
3. **Запустите** `install.bat` (двойной клик)
4. **Дождитесь** завершения установки
5. **Запускайте** программу через `run.bat`

## Что делает установщик:
- Проверяет наличие Python
- Создает виртуальное окружение  
- Устанавливает все зависимости
- Автоматически определяет браузеры
- Скачивает нужные драйверы

## Решение проблем:

**Если Python не установлен:**
- Скачайте с python.org (версия 3.8+)
- При установке отметьте "Add Python to PATH"

**Если браузер не определяется:**
- Установите Chrome или Yandex Browser
- Перезапустите install.bat
